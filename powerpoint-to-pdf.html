Frequently Asked Questions

What's the best way to convert PowerPoint to PDF? ▾
For the best visual quality, export your slides as PNG or JPEG images from PowerPoint (File → Export → Save as PNG/JPEG), then upload those images here. This creates a pixel-perfect PDF.
Can I convert directly from .pptx? ▾
Yes — use the PPTX mode to extract slide titles and body text from .pptx files. Note that this text-only mode does not reproduce images, shapes, animations or visual formatting from the original presentation.
Does my file get uploaded? ▾
Never. All processing happens in your browser. Your files never leave your device.
How do I export slides as images from PowerPoint? ▾
In PowerPoint: File → Export → Change File Type → PNG or JPEG. Or: File → Save As → choose PNG or JPEG format.
Can I reorder slides? ▾
Yes — in Slide Images mode, drag and drop the slide thumbnails to rearrange them before converting.
What slide ratios are supported? ▾
16:9 (widescreen), 4:3 (standard), auto-detect from image size, or fit to the image's own dimensions.
