What types of protection can it remove? ▼

Will this damage the PDF? ▼
No — only the password layer is removed. All content is preserved exactly.

What Happens During Unlocking

Unlocking is a decryption operation, not a password crack. You provide the correct password; pdf-lib decrypts the PDF content using standard AES-128/AES-256 PDF encryption algorithms; a clean unencrypted copy is saved for download. Your password is used locally — it is never sent anywhere. The original encrypted file on your device is untouched.

If a PDF has only an owner password (you can open it but can't print or copy), pdf-lib can often remove these permission restrictions without needing the owner password — because the content itself is unencrypted, only permission flags are set. If a PDF has a user password (required to open it), you must enter the correct password. After unlocking, re-protect with a new password using Protect Files if needed.
