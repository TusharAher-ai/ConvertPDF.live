Frequently Asked Questions

Can I edit the existing text in a PDF? ▼
Browser-based editing replaces or covers existing content with new layers. True text editing (changing a word in the original text) requires server-side PDF processing.
Will my annotations be visible to others? ▼
Yes — all annotations are flattened into the PDF on download, so they appear as permanent visible changes.
Can I undo annotations? ▼
While editing, use Ctrl+Z (or Cmd+Z on Mac) to undo recent actions before downloading.

How the PDF Editor Works Technically

PDF.js renders each page onto an HTML canvas. An annotation layer sits on top for interactive editing. When you save, pdf-lib flattens all annotations permanently into the PDF output. Highlight adds a semi-transparent colour rectangle. Text box places editable text (ideal for filling non-interactive form fields). Freehand drawing captures mouse or stylus strokes. Sticky note adds a collapsible comment. All annotations appear in any PDF viewer after download.

For a complete review workflow: highlight key clauses, add text box comments, circle issues with freehand drawing, then sign on the final page with the Sign PDF tool. Download, email back. No software, no account, no cost — done in minutes entirely in your browser.
